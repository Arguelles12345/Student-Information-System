# Requirements

## Functional Requirements
- The system must allow adding a student.
- The system must display all students in a table.
- The system must allow viewing full student details.
- The system must allow updating student information.
- The system must allow deleting a student.
- The system must support searching students by:
  - ID
  - Name
  - Course
  - Year Level
  - Email
  - Address
  - Gender
  - Birthdate (mm/dd/yyyy)
  - Guardian
  - Contact Number

## Non-Functional Requirements
- The system should respond quickly.
- The UI should be simple and user-friendly.
- The system should work without a database.
- The system should validate required inputs.
