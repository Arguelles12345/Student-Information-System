# Context

The Student Information System (SIS) is a web-based application designed to manage student records efficiently.

The system allows users (admin) to:
- Add new students
- View student list
- Search students
- Update student information
- Delete student records

This system does not use a database. Instead, it uses in-memory storage (arrays) in the backend.

The goal is to provide a simple and functional CRUD-based system for managing student data.
